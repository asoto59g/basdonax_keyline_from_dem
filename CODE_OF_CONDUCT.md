# Code of Conduct

This project follows a respectful, technical and collaborative working environment.

Contributors are expected to:

- Be respectful.
- Provide constructive feedback.
- Document technical decisions.
- Avoid personal attacks.
- Respect different professional backgrounds.
- Prioritize scientific and engineering rigor.

Unacceptable behavior includes:

- Harassment.
- Discrimination.
- Insults.
- Intentional disruption.
- Misrepresentation of technical results.
