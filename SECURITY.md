# Security Policy

## Supported Versions

| Version | Supported |
|---|---|
| 0.6.x | Yes |
| <0.6 | No |

## Reporting a Vulnerability

If you discover a security issue, please open a private report through the repository or contact the maintainer.

Please do not publish exploitable issues publicly before maintainers have reviewed them.

## Security Notes

This plugin:

- Does not require external network access.
- Does not execute remote code.
- Does not collect user data.
- Processes local GIS files selected by the user.

Users should verify all input datasets before running hydrological or engineering design operations.
